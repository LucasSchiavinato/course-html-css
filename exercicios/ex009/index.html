Código-fonte

O comando document.getElementById('TESTE') é escrito em linguagem Javascript, mas seu código não esta organizado a separar corretament as letras para melhor compreendimento de espaço, assim se adiciona a tag code.

document.getElementById('TESTE')


Exemplo abaixo em python, levando em consideração as tag pre para deixar o código identado e o code para deixar ele com espaços justificados.

        
            num = int(input('Digite um número'))
            if num % 2 == 0;
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num} é IMPAR')
            print('Fim do programa')
